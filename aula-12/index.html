O que é JavaScript?

JavaScript é a linguagem responsável por adicionar interatividade aos sites. Com ele, você pode criar animações, menus, pop-ups, validações e muito mais.

Por que aprender JavaScript?

  • Usado em praticamente todos os sites modernos.
  • Permite criar interfaces interativas.
  • É a linguagem principal do front-end.
  • Também pode ser usada no back-end (Node.js).

Exemplo Básico: If e Else

O if/else é como uma pergunta: “Se isso for verdade, execute tal ação. Caso contrário, execute outra.” 

let idade = 18;

if (idade >= 18) {
    console.log("Você é maior de idade");
} else {
    console.log("Você é menor de idade");
}
